# From FEM to Neural Operators
Drawing a parallel between learning a Differential Equation using Finite Element Method, PINNs, and Neural Operators


🌍 Overview
This repository accompanies the Spring School course:
[From FEM to Neural Operators: Scientific Machine Learning for Computational Mechanics](https://www.uni-weimar.de/springschool/de/16/from-fem-to-neural-operators-scientific-machine-learning-for-computational-mechanics/). Hosted at Bauhaus-Universität Weimar, this course explores a transformative shift in computational mechanics — from traditional Finite Element Methods (FEM) to Neural Operators and Scientific Machine Learning (SciML). Traditionally, FEM has been the backbone of solving partial differential equations (PDEs). Today, emerging AI-based approaches such as Physics-Informed Neural Networks (PINNs) and Neural Operators are redefining how we model, simulate, and accelerate physical systems. This repository is designed as a hands-on companion, guiding you through this transition.
